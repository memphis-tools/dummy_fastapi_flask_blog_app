DUMMY OPS - TECHNOS

WebApp Python déployée sur Docker Swarm. Docker Compose.

Frontend: Nginx, Flask, FastAPI. Certbot pour un certificat HTTPS gratuit. Hcaptcha pour le captcha.

Backend: Hashicorp Vault, Postgresql, Celery, RabbitMQ (Celery broker), Reddis (Celery db backend). Twilio SendGrid et Mailtrap pour adresser des emails.

Dépôt Gitlab (Github en mirroir sur dépôt public) : une chaine CI-CD est en place: tests, build, et publication si branche master.

Machine virtuelle créée chez DigitalOcean. Approvisionnement effectué avec Hashicorp Terraform.

Domaine réservé sur NameCheap. Gestion du consentement utilisateur: ConsentManager.

Connecté et protégé par Cloudflare.

Betterstack pour adresser des logs applicatifs (faire de la supervision, dont redondance partielle avec celle de DigitalOcean).

Codacy pour la qualité du code. TabNine et Cursor pour aider à la mise en place des tests. Usage de l'extension sur IDE VSCode

Aikido Security et Snyk pour vérifier les vulnérabilités niveau applicatif. Lynis niveau système.
